# Purrooser
Web browser with Python

## Bootstrap
Windows/DOS:
```
pip install -r requirements.txt
```

macOS/Linux/Unix:
```
pip3 install -r requirements.txt
```

## Run
Windows/DOS:
```
python -m src.main
```

macOS/Linux/Unix:
```
python3 -m src.main
```
