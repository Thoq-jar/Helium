# Purrooser
Web browser with Python

## Get
Any OS:
```
git clone https://github.com/Thoq-jar/Purrooser.git
```

## Bootstrap
Windows/DOS:
```
pip install -r requirements.txt
```

macOS/Linux/Unix:
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run
Any OS *(Make sure if on Unix-Like system you are in venv)*: 
```
python -m src.main
```
